int32_t  duration 
)

指定したWAVファイル（8-bit 8kHz mono）を再生する．今再生しているサウンドは停止される．

引数
p_memfileWAVファイルのメモリファイルへのポインタ
duration出力持続時間．単位：ミリ秒．SOUND_MANUAL_STOPを指定した場合は手動で停止しないと最後まで再生する．
戻り値
E_OK正常終了
